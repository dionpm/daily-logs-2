Daily Test Log — PWA (Preloaded Systems)
=====================================

This package is preloaded with your system names:
- Chilled Loop, Hot Loop
- Laundry Boiler 1–2, Food Boiler 1–2
- Cooling Tower 1–2
- Condensate 1–2

How to host (GitHub Pages)
--------------------------
1) Create a new repo on GitHub.
2) Upload these files at the repo root.
3) Settings → Pages → Deploy from branch (main, /root).
4) Open the site URL on your phone and "Add to Home Screen".

Local test
----------
python3 -m http.server 8000
# then open http://localhost:8000

Backups
-------
Use Export JSON to back up or migrate data between devices.
